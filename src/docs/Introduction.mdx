# Introduction to Rudiment UI

Rudiment UI is a template for building accessible React component libraries. It is built on [React Aria](https://react-spectrum.adobe.com/react-aria/) for accessibility and [Tailwind CSS 4](https://tailwindcss.com/) for styling.

## Who is it for?

Rudiment UI is for teams that want to start with a solid foundation and customize it for their brand. Instead of building a component library from scratch, you get:

- **Components** that are accessible out of the box
- **Design tokens** that make theming straightforward, including dark mode support
- **Tests** that verify behavior and accessibility
- **Documentation** powered by Storybook

## Quick start

1. **Clone the repo**

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start Storybook**

   ```bash
   npm run dev
   ```

4. **Customize design tokens:** edit the values in `tokens/tokens.json` (primitive values), `tokens/semantic.json` (theme-level decisions), and `tokens/themes/dark/semantic.json` (dark mode overrides)

5. **Build tokens**
   ```bash
   npm run build:tokens
   ```

6. **Toggle dark mode:** use the theme switcher in the Storybook toolbar, or set `data-theme="dark"` on your root element

## Using components

Import any component directly:

```jsx
import { Button, Input, Badge, Card, Icon, Stack } from 'rudiment-ui'
```

## Using styles

Include the stylesheet in your app entry point:

```js
import 'rudiment-ui/styles'
```
