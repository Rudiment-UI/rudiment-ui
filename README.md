# Rudiment UI

Template containing a complete, accessible React component library built on React Aria, styled with Tailwind CSS, documented in Storybook, and structured around a three-tier design token architecture. It includes both UI components (buttons, inputs, modals) and intrinsic layout primitives (Stack, Sidebar, Grid, and others) driven by layout tokens. The buyer clones the repo, swaps token values to match their brand, and has a working component library with documentation, tests, layout system, and accessibility baked in.
